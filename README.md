🎬 ScreenMatch | Java Backend

Simulação de uma plataforma de streaming de filmes e séries, desenvolvida em Java, com foco na aplicação prática de lógica de programação e Programação Orientada a Objetos (POO).

🚀 Sobre o projeto

O ScreenMatch é um sistema backend que simula funcionalidades básicas de uma plataforma de streaming, permitindo o gerenciamento e processamento de dados de filmes e séries.

Este projeto faz parte da minha jornada de transição de carreira de advogado para desenvolvedor backend, com foco em Java.

🎯 Objetivo

Consolidar conhecimentos em backend com Java e evoluir o projeto até um nível profissional, simulando cenários reais de desenvolvimento.

📌 Funcionalidades atuais
🎬 Cadastro de filmes e séries
🧱 Aplicação de herança e reutilização de código
⭐ Cálculo e exibição de avaliações
📊 Organização de dados utilizando ArrayList
🧠 Sistema de recomendação baseado em classificação
🔌 Uso de interface (Classificavel) para padronização de comportamento

🚧 Próximas evoluções
Implementar filtros de busc
Ordenação de conteúdos
Separação de responsabilidades (Service Layer)
Transformar em API REST com Spring Boot
Integração com banco de dados
